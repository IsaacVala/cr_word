Create and retrieve data types

Use this tool to create data types, or to explore data types that already exist on a worksheet.

To create a data type, use the select list to choose a data type (string, double, boolean, entity, web image, or formatted number), enter data in the displayed fields, and then select Set data. The data type will be inserted into the currently active cell on the worksheet.

To explore existing data types, select the cell on the worksheet that contains data, and then select the Get data button.
